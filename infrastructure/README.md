# Infrastructure

This folder is for cloud architecture notes, deployment steps, and security configuration.

Track:

- Cloud platform selected.
- Network diagram.
- Firewall or security group rules.
- IAM roles or managed identities.
- Secrets manager configuration.
- Public and private components.
- Before/after security changes.

Suggested evidence:

- Baseline architecture diagram.
- Secured architecture diagram.
- Before/after network rules.
- Before/after identity permissions.
- Before/after secret storage method.
